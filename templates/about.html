The game system HeXXen 1733™ is an adaption of the german Hexxen 1733 RPG for use in Foundry VTT. It provides character and item sheets, a dice tool as well as HeXXen-1733-Scriptorium content.

For the dice tool to work properly, the module special-dice-roller is required.
